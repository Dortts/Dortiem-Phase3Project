# SyriaTel Customer Churn Prediction

A classification project predicting customer churn using Logistic Regression, Decision Tree, and Random Forest models.

## Dataset
SyriaTel telecom customer data — 3,333 records, 20 features.

## Results
- Best model: Logistic Regression (Test Recall: 74.2%)
- Key churn drivers: customer service calls, day charges, international plan

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook
